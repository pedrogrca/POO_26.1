# 📚 POO_26.1 - Programação Orientada a Objetos

Bem-vindo(a) ao repositório da disciplina de **Programação Orientada a Objetos (POO)** do **Instituto Federal do Rio Grande do Norte (IFRN)**! 🎓

Este espaço é dedicado a armazenar todos os exercícios, projetos, resoluções de questões e anotações desenvolvidos ao longo do curso. O objetivo principal é documentar a evolução e o aprendizado prático nos conceitos fundamentais de orientação a objetos.

---

## 🚀 Tecnologias e Paradigmas

* **Linguagem:** Python 🐍
* **Paradigma:** Orientação a Objetos (Classes, Objetos, Herança, Polimorfismo, Encapsulamento e Abstração)
* **Ambiente e Versionamento:** Git & GitHub

---

## 📂 Estrutura do Repositório

O repositório está organizado por tópicos e unidades da disciplina para facilitar a navegação. Conforme o semestre avança, novas pastas serão adicionadas. 

Atualmente, a estrutura conta com:

* 📁 **`Introdução a POO/`** - Exercícios básicos, modelagem inicial, definição de classes, atributos e métodos (inclui a Questão 4).
* ⏳ *(Novas pastas de herança, polimorfismo, etc., serão adicionadas conforme o andamento das aulas...)*
